----- Projet Langages du Web -----

--- Précédentes expériences : 

-- CSS
- Bootstrap
- Flexbox

-- JS
- JQuery
- React.js/Vue.js


--- Technologies utilisées pour ce projet : 

- Foundation (SCSS)
- Foundation Icons (SVG)
- JQuery (pour faciliter les appels AJAX)
- JavaScript natif
- JSON
- HTML5/CSS3


--- Pré-requis : 

- Lancer un serveur Web pour afficher la page, faute de quoi les appels AJAX ne seront pas fonctionnels (commande php -S 0.0.0.0:(port) dans le dossier du projet pour un terminal UNIX/macOS, ou bien WAMP sous Windows, afin d'accéder à la page via l'URL http://localhost:(port) )


-- Remarques :

- Les sources de Foundation sont situées dans le dossier "Projet"

- J'ai volontairement réduit la liste des éléments importés dans le fichier app.scss, afin  de ne pas charger les éléments que je n'utilisais pas, permettant au passage de réduire la taille du fichier CSS final.

- Mes styles personnalisés sont dans un mixin, situé dans le fichier Projet/bower_components/foundation-sites/scss/util/_ownstyles.scss

- Ils sont ensuite importés, puis inclus dans le fichier Projet/scss/app.scss
